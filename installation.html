Linux

Ruvchain is well supported on Linux. There are a number of distribution packages and additional installation instructions available:

  • Debian, Ubuntu, elementaryOS, Linux Mint and similar
  • Red Hat Enterprise Linux, Fedora, CentOS and similar
  • Gentoo Linux
  • OpenWrt
  • Manual install or build from source

macOS

Ruvchain is well supported on macOS. The preferred installation method is the .pkg installer:

  • macOS .pkg installer
  • Manual install or build from source

Windows

Ruvchain works on Windows and an MSI installer is available, but the installer is supported on a best-effort basis only and is not well tested.

Android

Ruvchain is supported on Android with the following applications:

  • Ruvchain for Android, reference client based on the v0.4 release candidate
  • Crispa for Android, contributed by the community

FreeBSD

Ruvchain is well supported on FreeBSD and an official net/ruvchain port is available.

Ubiquiti EdgeRouter

Ruvchain is well supported on EdgeOS 2.x and packages are available.
